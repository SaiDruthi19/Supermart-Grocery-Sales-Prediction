# Supermart Grocery Sales Prediction

**Project Overview:**  
Analyzed and predicted grocery sales using historical supermarket data to help forecast future sales and understand trends.

**Highlights:**  
- Performed **data cleaning** and handled missing values.  
- Conducted **Exploratory Data Analysis (EDA)** to visualize sales trends across categories, cities, and months.  
- Built a **Ridge Regression model** to predict sales.  
- Compared **actual vs predicted sales** for model evaluation.  
- Generated **final predictions** in an Excel file for easy reference.  

**Technologies:**
- **Python:** Pandas, NumPy  
- **Machine Learning:** Scikit-learn (Ridge Regression)  
- **Visualization:** Matplotlib, Seaborn  
- **Data Output:** Excel (`.xlsx`)  

**How to Use:**  
1. Open `Supermart_Grocery_Sales_Prediction.ipynb` in Jupyter Notebook or Google Colab.  
2. Run the notebook step by step to explore analysis, visualizations, and predictions.  

**Files in Repository:**  
- `Grocery_sales.ipynb` – Full project notebook.  
- `Final_Predictions.xlsx` – Predicted sales results.
